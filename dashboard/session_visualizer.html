Standalone visualizer merged into the integrated app

The legacy SessionVisualizer class (visualizer.js, 0401fbde) is superseded by the modular dashboard: 3D pool scene, accel/gyro/mag charts, phase-colored trail, entry angle, ‖ω‖ at water entry, haptic markers, Wi-Fi sync, merge, and ideal stroke.

Full dashboard: from the dashboard folder run:

python3 wifi_session_processor.py

Then open http://localhost:8844/ (or set PORT / --port).

USB session logger only. session_logger.py still captures JSON. Use the integrated dashboard after syncing from the wearable, or the bundled visualizer.js route for legacy file playback.
