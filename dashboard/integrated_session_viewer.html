Wi‑Fi sync (wearable has no dashboard)
  1. Hold sync on the wearable — it opens the GoldenForm Wi‑Fi network (same for first link or after a swim).
  2. Join Wi‑Fi — connect this phone/laptop to that network (stay on it for sync).
  3. Tap “Sync Device” below — sessions download; 0 sessions is valid before your first swim (link + registry + profile still apply).
  4. After swimming — stop recording on the device, enter sync again, repeat steps 1–3 to fetch new sessions.

Add the next device
  1. One Wi‑Fi at a time — only power / sync this unit so your laptop joins the correct GoldenForm network.
  2. Connect — join that Wi‑Fi, then use Load from connected device below (or type the HW ID printed on the unit / in serial logs).
