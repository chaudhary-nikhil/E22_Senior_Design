Your bands

The app reads the hardware ID from the band after you connect. Two bands: use one Wi‑Fi at a time and add each separately.

IMU calibration (sensor fusion)

Stay on the band Wi‑Fi here. Numbers update live. While you swim, the radio stays off to save power; stroke cues use the motor on the band (faster than sending over Wi‑Fi). For downloads, use sync mode again and run Session Sync now like usual.

Each channel is 0 to 3 while you are connected. Open the guide for the figure eight and face tilts. We save a snapshot in this browser when quality goes up (you can save manually too).
