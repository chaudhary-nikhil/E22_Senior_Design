Upload an above-water lab video. The server runs MediaPipe Hands and exports wrist position over time (normalized + pixels). Pair the JSON with your raw IMU session to compare shapes and tune integration.

Requirements

On the computer running the dashboard, install:

pip install -r requirements-calibration.txt

Then restart wifi_session_processor.py. Max upload size ~120 MB.
